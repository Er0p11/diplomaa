# Менеджер метилирования (Methylation Manager)

Десктопное приложение для хранения, обработки, анализа и корректировки экспериментальных данных метилирования генов ДНК. Разработано в рамках выпускной квалификационной работы (ВКР).

## Описание проекта

В исследовательской практике экспериментальные замеры метилирования ДНК часто содержат искажения, вызванные аппаратными и методическими погрешностями. Данная программа предоставляет удобный графический интерфейс для:
- Ведения локальной базы данных (SQLite) со справочниками генов, праймеров, исследователей и реактивов.
- Хранения калибровочных (эталонных) и искаженных измерений.
- Автоматического построения аппроксимационных функций (кубическая, гиперболическая со сдвигом, комбинированная).
- Визуализации данных и графиков аппроксимаций.
- Восстановления (коррекции) истинных значений метилирования из искаженных "сырых" данных.

## Стек технологий

- **Язык**: Python 3.9+
- **GUI Фреймворк**: PySide6 (Qt for Python)
- **База данных**: SQLite (через модуль `sqlite3` и `QSqlDatabase`)
- **Вычисления и оптимизация**: NumPy, SciPy (`scipy.optimize`)
- **Визуализация**: Matplotlib (с поддержкой рендеринга LaTeX-формул)

## Структура проекта

- `main.py` — Точка входа в приложение, логика главного окна и переключения рабочих пространств.
- `tabs.py` — Реализация графических вкладок (Калибровка, Коррекция, Редактор БД, Справка).
- `db.py` — Утилиты для работы с БД, инициализация SQL-схемы (10 нормализованных таблиц).
- `approx.py` — Математическое ядро: функции аппроксимации, расчет метрик (СКО, относительная ошибка), инвертирование функций.
- `compute.py` — Логика автоматического пересчета усреднений и обновления коэффициентов в БД.
- `seed_db.py` — Скрипт для генерации тестовой базы данных с реалистичными значениями.
- `utils.py` / `formatters.py` — Вспомогательные функции парсинга CSV/TXT и форматирования вывода (числа, LaTeX).
- `LOADER.py` — Модуль автообновления приложения (Auto Updater).

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИК/methylation-manager.git
   cd methylation-manager
   ```

2. Установите необходимые зависимости. Рекомендуется использовать виртуальное окружение:
   ```bash
   pip install PySide6 matplotlib numpy scipy
   ```

3. (Опционально) Сгенерируйте тестовую базу данных для ознакомления с функционалом:
   ```bash
   python seed_db.py
   ```
   *В корневой папке появится файл `sample_db.sqlite3`.*

4. Запустите приложение:
   ```bash
   python main.py
   ```

## Использование

1. При первом запуске выберите **«Создать новую базу»** (или откройте существующую/сгенерированную тестовую).
2. На вкладке **«Редактор БД»** заполните базовые справочники (Гены, Исследователи и т.д.).
3. На вкладке **«Гены и Калибровка»** вносите экспериментальные точки вручную или через импорт CSV. Программа автоматически рассчитает аппроксимации и построит график.
4. На вкладке **«Коррекция»** загрузите ваши реальные искаженные замеры, выберите лучшую аппроксимационную модель, и программа рассчитает исправленные (истинные) значения.
