# Описание программного комплекса "Менеджер метилирования"

## 1. Цели и задачи приложения

**Цель:** Автоматизация процесса обработки и анализа данных метилирования ДНК, полученных в ходе лабораторных исследований.

**Задачи:**
1.  **Централизованное хранение данных:** Создание единой базы данных для хранения информации о генах, исследователях, публикациях, реактивах и результатах измерений.
2.  **Калибровка измерений:** Обеспечение возможности ввода калибровочных данных (зависимость истинного уровня метилирования от измеренного прибором).
3.  **Математическое моделирование:** Автоматический расчет аппроксимирующих функций (калибровочных кривых) для каждого гена с оценкой точности моделей.
4.  **Коррекция данных:** Исправление новых экспериментальных данных на основе построенных калибровочных моделей для получения истинных значений метилирования.
5.  **Визуализация:** Построение наглядных графиков калибровочных кривых и результатов коррекции.
6.  **Экспорт результатов:** Возможность выгрузки обработанных данных в формате CSV для дальнейшего использования в научных публикациях и отчетах.

## 2. Описание возможностей и функций

Приложение реализовано на языке Python с использованием библиотеки PySide6 (Qt) для графического интерфейса и SQLite для хранения данных.

### Основные функциональные блоки:

#### А. Управление базой данных (Вкладка "Редактор БД")
*   **Просмотр и редактирование:** Полный доступ ко всем таблицам базы данных (Гены, Исследователи, Публикации и др.).
*   **Добавление записей:** Удобные диалоговые окна для внесения новой информации.
*   **Удаление записей:** Возможность удаления устаревших или ошибочных данных.

#### Б. Калибровка (Вкладка "Гены и Калибровка")
*   **Выбор гена:** Работа с конкретным генетическим маркером.
*   **Ввод калибровочных точек:** Внесение пар значений "Истинное" (True) vs "Измеренное" (Observed).
*   **Импорт данных:** Загрузка больших массивов калибровочных данных из CSV/TXT файлов.
*   **Автоматический расчет:** Приложение автоматически строит несколько математических моделей (линейная, кубическая, гиперболическая) и выбирает лучшие коэффициенты.
*   **Оценка точности:** Для каждой модели рассчитывается среднеквадратическое отклонение ($\sigma$) и относительная ошибка ($\epsilon$), что позволяет исследователю выбрать наиболее точную модель.

#### В. Визуализация (Вкладка "Графики")
*   **Построение кривых:** Графическое отображение калибровочных точек и аппроксимирующих функций.
*   **Сравнение моделей:** Возможность визуально оценить, насколько хорошо выбранная математическая модель описывает экспериментальные данные.

#### Г. Коррекция данных (Вкладка "Коррекция")
*   **Выбор модели:** Пользователь выбирает ген и наиболее подходящую аппроксимирующую функцию.
*   **Ввод экспериментальных данных:** Вставка "сырых" значений, полученных с прибора.
*   **Расчет истинных значений:** Приложение использует обратную функцию выбранной модели для пересчета "сырых" данных в истинные значения метилирования.
*   **Визуальный контроль:** Отображение графика коррекции, где показано положение новых точек на калибровочной кривой.
*   **Экспорт:** Сохранение скорректированных данных в файл.

## 3. Иллюстрации работы приложения

*(В этот раздел необходимо вставить скриншоты)*

**Скриншот 1: Вкладка "Гены и Калибровка"**
*Пояснение:* Здесь исследователь выбирает ген и вводит эталонные значения для построения калибровочной кривой. В правой части отображаются рассчитанные формулы аппроксимации с оценкой погрешности.

**Скриншот 2: Вкладка "Графики"**
*Пояснение:* Визуализация калибровочной кривой. Синие точки — усредненные экспериментальные данные, линия — математическая модель.

**Скриншот 3: Вкладка "Коррекция"**
*Пояснение:* Процесс обработки новых данных. Слева введены "сырые" значения, справа в таблице — пересчитанные истинные значения. На графике красными точками отмечены результаты коррекции.

**Скриншот 4: Диалог добавления точки**
*Пояснение:* Форма для ручного ввода калибровочных значений.
